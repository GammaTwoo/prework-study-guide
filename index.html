Prework Study Guide

HTML

  • the head element contains information about the webpage
  • the body element represents the visible content shown to the user
  • doctype establishes the language of the document
  • lang establishes the language (en for english)
  • Title establishes the text in the tab, i think
  • img tells the website to show an image, src tells the website where to get it from
  • alt describes images using text (mostly for screen readers)
  • section denmominates an area of code
  • h1-h6 are different degrees of header, 1 being biggest (1 per page) and 6 being smallest
  • ol means ordered list (numbered) and ul means unordered list (bullets)

CSS (Cascading Style Sheets)

  • there are 3 ways CSS can be used: inline CSS, internal CSS, and external CSS
  • inline CSS uses style elements on the lines that youre using them
  • internal CSS uses style elements in the head section of the html doc
  • external CSS give CSS rules in a separate file (makes design changes easier)

Git

  • git status: shows current branch
  • git checkout -b [branch name]: creates a new branch and switches to it
  • git checkout: switches to an existing branch
  • git add: adds all modified and new files in the current directory and all subdirectories to the staging area
  • git commit -m "[message]": captures a snapshot of currently staged changes
  • git pull: grabs and downloads content from a remote repo and updates the local repo to match
  • git push: uploads local repo content to a remote repo
